## NEWSFINDER



## Описание
Новостной сайт с поиском по ключевым словам и добавлением статей в избранное. 
Новости загружаются с помощью NewsAPI, избранное хранится в localStorage браузера.

## Функционал
- Просмотр новостей по любому слову;
- Просмотр заголовка и краткого описания новости;
- Добавление новостей в избранное (с защитой от дублирования);
- Избранное сохраняется после перезагрузки страницы.

## Технологии
- Node.js + express + axios + dotenv (бэкенд);
- NewsAPI (источник новостей);
- HTML, CSS, JavaScript (фронтенд);
- localStorage (хранение избранного).



### Как скачать и запустить

1. Скачать проект с GitHub:
Написать в командной строке:
`git clone https://github.com/inlaazyyy/special-journey`
`cd special-journey`

2. Установить библиотеки:
Написать в командной строке:
`npm install`

3. Получить API-ключ:
Зарегистрироваться на https://newsapi.org,
Скопировать ваш API-ключ.

4. Настроить окружение:
Создать файл *.env* на основе *.env.example*,
Вставить ваш API-ключ в NEWS_API_KEY = вставьте_сюда_свой_ключ

5. Запустить сервер:
Написать в командной строке:
`npm run dev`

6. Открыть в браузере ссылку:
http://localhost:8000
